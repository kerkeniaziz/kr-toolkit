== Description ==

KR Toolkit is the essential companion plugin for KR Theme. It provides:

* One-click demo import
* Child theme management
* Customization options
* License management
* System requirements checker

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/kr-toolkit/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to KR Toolkit in the admin menu to get started

== Frequently Asked Questions ==

= Does this work with any theme? =

No, KR Toolkit is specifically designed to work with KR Theme.
